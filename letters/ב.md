# ב — בית / Forward Deepening

Status: spec synced with `mission-statement.md`. Implementation may differ.

## Definition (from mission-statement)

- **Select:** current focus target (x).
- **Bound:** create a new house-frame around the target with a forward-only opening (conceptually: three sealed faces, one open face).
  - If execution is at word-entry baseline (no constructed referent yet), first allocate a minimal seed referent `X0`, then house `X0`.
- **Seal:** enter the newly created house-frame (`Omega := House(x)`) and set focus to it.

Operationally this is a **non-idempotent deepening** step:

- `ב(x) -> House(x)`
- `בב(x) -> House(House(x))`
