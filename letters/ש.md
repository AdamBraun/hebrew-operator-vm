# ש — שין (tilted ן spine + two ו attached at the terminal end)

Status: spec synced with `mission-statement.md`. Implementation may differ.

## Definition (from mission-statement)

- **Select:** current focus (`F`) - the handle forwarded by the preceding letter.
- **Bound** (parameterized by direction):
  - **External** (`שׁ`, default undotted): allocate three outward-facing ports from `F` (`spine`, `left-handle`, `right-handle`). All three inherit `F`'s effective constraint bundle. These are selectable by downstream letters.
  - **Internal** (`שׂ`): subdivide `F`'s interior into three compartments (same topological shape, pointed inward). Internal bracing produces carry resolution on `F`'s `cont`-chain: the structural support that `ס` achieves with a single `supp` edge, internal fork achieves by creating a braced interior that resolves open carries within its scope.
- **Seal:** replace focus with the resulting structured handle. Pass thread forward (next letter selects from this handle).

Dot selection:

- Right dot -> `direction=external` (default)
- Left dot -> `direction=internal`
- Undotted -> `direction=external` (backward compatibility)

Milui:

- `שי״ן = fork (ש) -> pin (י) -> straighten (ן)`.
- After forking (either direction), the name describes the resolution path: commit to one pinned line and lock it as continuation.
