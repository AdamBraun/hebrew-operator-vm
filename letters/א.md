# א — Alias-Anchor

Status: spec synced with `mission-statement.md`. Implementation may differ.

## Definition (from mission-statement)

- **Select:** take the word-entry focus `F0`; the construct operand is implicit.
- **Bound:** if the word has no current construct (`C = ∅`), allocate a fresh word-local handle `h := alloc()` and use `h` as construct; otherwise use the current construct `C` as-is.
- **Seal:** commit `Alias(F0, C)` with bidirectional transport links and keep `C` as the output focus for following letters.

Operational rule:

- If `C = ∅`: `C := alloc(); Alias(F0, C)`.
- Else: `Alias(F0, C)` and do not replace `C`.
