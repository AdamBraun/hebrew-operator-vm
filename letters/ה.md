# ה — Exposure / announcement

Status: spec synced with `mission-statement.md`. Implementation may differ.

## Definition (from mission-statement)

- **Select:** the currently focused targets/rules.
- **Bound:** mark selected bounds as “visible/public/declared”.
- **Seal:** commit to the public layer (like writing on the board): later rules can reference this as a named, time-stamped declaration.
- **Mode semantics (`H(mode)`):**
  - `public` (default): standard declaration, returns a declaration handle.
  - `breath` (HH): word-final fallback without mappiq; no new declaration handle, only tail shading on current focus.
  - `pinned` (HY): execute `public`, then export a pinned handle.
  - `alias` (HA): execute `public`, then return an alias/transport bind to the pre-he focus.
- **Mappiq (`הּ`):** forces `H(mode)=pinned` and prevents final-heh breath degradation.
