# ק — Bare head with detached adjunct leg

Status: spec synced with `mission-statement.md`. Implementation may differ.

## Definition

- **Select:** current construct `X`. If no construct exists yet in the word, select word-entry ambient `W₀`.
- **Bound:**
  1. `h := alloc()`
  2. add `head_of(h, X)`
  3. add `carry(X, h)`
  4. `ℓ := alloc()`
  5. add `cont(h, ℓ)`
  6. add `carry(h, ℓ)`
  7. export `ℓ` as an adjunct of `h`
- **Seal:** set `F := h`.

## Interpretation

An unresolved head that also issues a detached, hanging act-line.

## Family

- `ר`: head only, unresolved
- `ד`: head only, resolved
- `ק`: head + detached leg, unresolved
- `ה`: head + detached leg, resolved

## Signature

- Arity: req 1, opt 0.
- Operand kinds: any.
- Selection precedence: default (see `spec/60-VM.md`).
- Distinctness/reflexive: default (see `registry/letters.yaml`).
